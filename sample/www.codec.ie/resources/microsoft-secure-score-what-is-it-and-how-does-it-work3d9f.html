Microsoft Secure Score: What is it and how does it work?

Written by Codec | May 3, 2023 11:46:22 AM
What is a Microsoft secure score?

Microsoft secure score is a measurement tool that compares an organisation's security stance against industry benchmarks and Microsoft's security recommendations. A higher score means better security and helps measure progress over time. Secure Score provides personalised recommendations to improve an organisation's security score and prevent cyber threats. This tool encourages users to adopt best practices and implement recommended security features, such as enabling multi-factor authentication and updating software regularly. It also alerts users of potential risks and provides guidance on how to resolve those risks.
