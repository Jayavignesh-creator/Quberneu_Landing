Codec are pleased to offer a number of Azure Immersion Workshops

Codec are pleased to offer a number of new dates for our Azure Immersion Series Workshops. Learn from Azure solution specialists at these interactive workshops which feature hands-on labs. Choose from Windows Virtual Desktop, Infrastructure Migration and/or .Net Modernization.

Windows Virtual Desktop: 25th Feb 2021 (10:00 – 15:00)

This workshop will help participants see the value of Windows Virtual Desktop to their business, remote work, and other user scenarios. For IT Pros (TDM, IT Implementor), this workshop provides participants with both the knowledge and skills to accelerate designing and implementing a virtual desktop infrastructure on Azure. More Details.

Infrastructure Migration: 11th March 2021 (10:00 – 15:00)

Helps participants understand how to migrate workloads to Azure. Provides the knowledge and skills to migrate on-premises infrastructure such as Windows Servers and Linux Servers, running line of business applications and their associated database tier to Azure. Audience: TDMs, IT Pros. More Details
